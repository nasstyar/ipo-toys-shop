# Django Store Project (Магазин игрушек для детей)

## Описание
Лабораторный проект на Django, демонстрирующий базовую настройку веб-приложения для интернет-магазина. Сайт похож на магазин Funtastik (https://www.funtastik.by/) по продаже детских игрушек.
